# Engine pre-cleaner performance

## Description
I apply data science skillsets in analyzing engine performance data.

### Motivation

This is a heavy-industry example of applying statistics (T-test, effect size) to evaluate claims from a supplier of engine precleaners performance enhancement on C-175 off highway truck (mining truck).
I was responsible for:
1. Identifying suitable trucks.
2. Feature selection.
3. Coordinating with others for the data structure and pipeline.
4. Analyzing the collected data.
5. Combining results on different claims / indicators.

## What's is shown?:
1. Application of DASK (pandas alternative to very big datasets).
2. Managing the data and performing T-test.
3. Performing effect size test (Cohen D's).
4. Analyzing the claims on the product given the setup for this comparative test.
